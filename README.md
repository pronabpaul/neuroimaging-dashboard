# Neuroimaging Dashboard - Interactive Region-Growing Segmentation

[![Live Demo](https://img.shields.io/badge/Demo-Hugging%20Face-blue)](https://huggingface.co/spaces/Pronabpaul/neuroimaging-dashboard)

Browser-based multi-cue region-growing tool for segmenting tumours, lesions, and haemorrhages in medical images (MRI, CT, PET).  
**Optimal segmentation is achieved when the seed point is placed on a pixel that is representative of the target region’s intensity.**  
The algorithm combines intensity, texture, gradient, direction consistency, and statistical similarity to accurately delineate boundaries, with real-time parameter adjustments.
